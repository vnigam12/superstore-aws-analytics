# Superstore AWS Analytics Project

## Architecture
IAM → S3 → Glue → Athena → QuickSight

## Dataset
Sample Superstore dataset

## Features
- ETL pipeline using AWS Glue
- Query layer using Athena
- Dashboard using QuickSight

## Setup Instructions
1. Upload data to S3
2. Run Glue crawler / ETL
3. Query via Athena
4. Build QuickSight dashboard
